Welcome to the Anarchist Code Club

A collection of writings on technology, anti/capitalism and utopia.

Why anarchism?

Anarchy is a simple, blog-aware, static site generator perfect for personal, project, or organization sites. Think of it like a file-based CMS, without all the complexity. Anarchy takes your content, renders Markdown and Liquid templates, and spits out a complete, static website ready to be served by Apache, Nginx or another web server. Anarchy is the engine behind GitHub Pages, which you can use to host sites right from your GitHub repositories.

Getting started

  1. Make sure you have Anarchy up and running
  2. Clone this repository
  3. Customize it
  4. Push to gh-pages
